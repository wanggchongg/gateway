此版本包含新功能：
1. 接收socket报文函数改为多线程，包括tcp(可以同时处理多条连接)和udp
2. 增加socket接收缓冲区
3. 发送socket报文函数改为多线程，目前tcp或udp分别只能发送一个ip号
4. 接收串口报文函数改为多线程
5. 增加串口报文缓存区和串口命令缓存区
6. 增加选择发送一个IP地址还是两个IP地址
7. 将UDP和TCP的发送都改为线程
8. 原来脚本调用串口设置函数改为recvPacFromSerial函数中调用串口设置函数
9. 增加udp的报文写入本地文件的线程，不过要求报文为可打印的ASCII字符,能正确的存入文本并可以正确地发送至socket。
